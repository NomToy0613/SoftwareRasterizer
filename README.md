# SoftwareRasterizer
**タイトル**
CPU Software Rasterizer

**概要**
CPU上で3Dグラフィックスパイプラインを自前実装した
ソフトウェアラスタライザです。
GPUのレンダリングパイプラインをブラックボックスとして扱わず、
内部処理を理解することを目的として実装しました。

**実装項目**
・3次元ベクトル（Vector3）
・4×4行列（Matrix4x4）
・クォータニオン（Quaternion）
・ワールド変換（Transform）
・ビュー変換（Camera）
・射影変換（Projection）
・パースペクティブディバイド
・ビューポート変換
・三角形ラスタライズ
・重心座標補間
・Zバッファによる深度管理
・フレームバッファ（pixelBuffer）
・Win32（GDI）を用いた描画
 
**今後の拡張予定**
*レンダリング機能*
・ニアクリッピング
・パースペクティブコレクト補間
・テクスチャマッピング
・ライティング
・DirectX11との表示統合
*エンジン機能*
・シーン管理
・リソース管理
・入力システム
・コンポーネントベースアーキテクチャ
・レンダリングパイプラインの抽象化

本プロジェクトは、自作CGエンジンの開発を最終目標とし、その基盤となるレンダリングパイプラインの理解と実装を目的として取り組んでいます。
